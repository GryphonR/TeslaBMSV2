◆ getAverageV()

float BMSModule::getAverageV ( )

Calculates the average cell voltage of the module.

This function loops over all 6 cell voltage values and averages the ones that are within the valid range (IgnoreCell to 60.0V). It keeps track of the number of valid cells and if the number of valid cells changes, it increments a miss counter. If the number of valid cells is the same for 3 consecutive reads, it resets the miss counter and sets the number of valid cells to the current value.

Returns
The average cell voltage of the module in Volts

Reads the status of the board to identify any flags, will be more useful when implementing a sleep cycle.

Reads the Alert Status, Fault Status, COV Faults, CUV Faults, and Balance Time from the module and stores them in the alerts, faults, COVFaults, CUVFaults, and balanceTime variables respectively.

◆ setIgnoreCell()

void BMSModule::setIgnoreCell ( float  Ignore)

Sets the voltage level below which a cell is ignored during balancing.

This is useful for cells that are not connected to the BMS (e.g. if you have a 12S BMS and you are using 10S or 14S batteries). You can set the ignore voltage to a value that is below the lowest cell voltage of the unused cells, so that the BMS does not try to balance the unused cells.

Parameters
IgnoreThe voltage level (in Volts) below which the cell is ignored.
