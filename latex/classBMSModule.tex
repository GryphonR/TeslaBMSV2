\doxysubsubsection{\texorpdfstring{getAverageV()}{getAverageV()}}
{\footnotesize\ttfamily float BMSModule\+::get\+AverageV (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Calculates the average cell voltage of the module. 

This function loops over all 6 cell voltage values and averages the ones that are within the valid range (Ignore\+Cell to 60.\+0V). It keeps track of the number of valid cells and if the number of valid cells changes, it increments a miss counter. If the number of valid cells is the same for 3 consecutive reads, it resets the miss counter and sets the number of valid cells to the current value.

\begin{DoxyReturn}{Returns}
The average cell voltage of the module in Volts 
\end{DoxyReturn}
